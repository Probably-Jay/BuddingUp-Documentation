◆ ValidateHash()

static bool SaveSystemInternal.SaveDataSerialiser.ValidateHash ( SaveGameData  data)
static

Validates if the checksum hash based on the current data matches the one stored with this data when it was last serialised. This should remain the same if calculated on the same data and so is used to make sure file has not been altered / corrupted. This will only be meaningful just after a strucutre is deserialised from a file. SaveGameData.hash is unaltered by this function.

Parameters
dataThe structure to be validated
Returns
If the hashes match (the data is the same and not-corrupted)
