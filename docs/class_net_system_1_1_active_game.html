IEnumerator NetSystem.ActiveGame.JoinOpenGameGroup ( GroupWithRoles  groupToJoin,
APIOperationCallbacks< NetworkGame >  resultsCallback 
)

Joins an open game on the server that was started by another player. This removes the title entity from the game and sets the game to "closed". Upon completion will invoke one of the following callbacks :

APIOperationCallbacks<NetworkGame>.OnSucess: The game was sucessfuly entered, its metadata was gathered, and a new NetworkGame object has been stored

APIOperationCallbacks<NetworkGame>.OnFailure: The call failed due to a networking error, or for one of the following reasons (retuned in callback):

  • FailureReason.TooManyActiveGames The user is a member of too many games and is not allowed to enter a new one
