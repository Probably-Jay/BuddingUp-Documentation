IEnumerator NetSystem.MatchMaker.CreateGame ( APIOperationCallbacks< NetworkGame >  resultsCallback)

Creates a new open game on the server, which will contain the calling client and the title entity. Upon completion will invoke one of the following callbacks :

APIOperationCallbacks<NetworkGame>.OnSucess: The game was sucessfuly created and entered, its metadata was gathered, and a new NetworkGame object has been stored

APIOperationCallbacks<NetworkGame>.OnFailure: The call failed due to a networking error, or for one of the following reasons (retuned in callback):

  • FailureReason.TooManyActiveGames The user is a member of too many games to enter a new one
  • FailureReason.AboveOpenGamesLimit The user cannot create a new game as there are no open games to join and they own too many open games to start a new one
