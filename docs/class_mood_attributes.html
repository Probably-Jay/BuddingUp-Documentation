string MoodAttributes.GetName ( Scales  scale)

Parameters
scaleThe scale the atribute is in
Returns
Returns the printable names for the provided scale at the current value of that scale. If the Pleasance scale holds a value of -3, it will return “Unpleasant”. However, if it held a value of 0 it would return “Neutral”.
