◆ EnterNewGame()

void NetSystem.NetworkHandler.EnterNewGame ( APIOperationCallbacks< NetworkGame >  parentCallbacks)

Enters new game. Will join an open game from the server if one is available, or will create a new game and publish it to the server. Upon completion will invoke one of the following callbacks :

APIOperationCallbacks<List<PlayFab.GroupsModels.GroupWithRoles>>.OnSucess: A game was sucesfully joined. Will return the NetworkGame that was joined

APIOperationCallbacks<List<PlayFab.GroupsModels.GroupWithRoles>>.OnFailure: The call failed due to a networking error, or for one of the following reasons (retuned in callback):

  • FailureReason.TooManyActiveGames The user is a member of too many games and is not allowed to enter a new one

  • FailureReason.AboveOpenGamesLimit There are no open games on the server to join, but the user cannot create a new game as they own too many open games already

◆ GatherAllMemberGames()

void NetSystem.NetworkHandler.GatherAllMemberGames ( APIOperationCallbacks< ReadOnlyCollection< NetworkGame >>  callbacks)

Gathers all games this client is a member of, including unstarted open games. This data is cached. Upon completion will invoke one of the following callbacks :

APIOperationCallbacks<List<PlayFab.GroupsModels.GroupWithRoles>>.OnSucess: The cache was hit, or the open games were sucessfully gathered along with their metadata and cached

APIOperationCallbacks<List<PlayFab.GroupsModels.GroupWithRoles>>.OnFailure: The call failed due to a networking error, or for one of the following reasons (retuned in callback):

  • FailureReason.PlayerIsMemberOfNoGames The player is not a member of any games. This is returned as an explicit error case to be handled
